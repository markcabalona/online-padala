# Quickstart
Start the server by running the following commands.

```bash
git clone https://github.com/markcabalona/online-padala.git
```
```bash
cd online-padala
```
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```
```bash
pip install -r requirements.txt
```
```bash
uvicorn main:app
```
